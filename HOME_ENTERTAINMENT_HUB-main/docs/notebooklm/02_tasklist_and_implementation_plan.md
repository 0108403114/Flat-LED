# タスクリストと実装プラン

最終更新: 2026-05-09

## 1. 進め方の前提

このプロジェクトは、いきなり完成形を作るのではなく、MVP でボトルネックを先に潰す方針です。

- まず 2 台構成でコアロジックを成立させる
- その後にシーン品質、外部制御、長時間安定性、多ノード化へ広げる
- 実機で確認済みの項目だけを「完了」とみなす

## 2. フェーズ進捗サマリ

### Phase A: 文書と骨組み

完了済み。

- 仕様書、ハード要件、ソフト設計、実装計画を作成
- BOM と電源方針を整理
- GPIO 割り当て案と UART 最小仕様を定義

### Phase B: ハードウェア調達と電源検証

一部進行中。

- ESP32-S3-Zero 2 台は調達済み
- CIO SMARTCOBY Pro SLIM 35W は調達済み
- ただし完成形電源チェーンの段階負荷試験は未完了

### Phase C: 中央ノード単体

未完了。

- マイク、ボタン、TFT、センサ、UI は今後の対象
- 完成形の HomeLiveHall コントローラ実装は未着手に近い

### Phase D: LED ノード単体

主要部は完了。

- PlatformIO + FastLED 構成を確立
- ESP32-S3-Zero で LED マトリクス駆動を確認
- WS2812B 実機制御を確認
- 安全系の基礎実装を導入

### Phase E: 通信統合

完了。

- UDP マルチキャスト受信
- 署名チェック、CRC、Target 振り分け
- 歌詞チャンク受信と再構成
- 差分更新と Force Reset
- 2 台同時動作 MVP

### Phase F: シーン統合と演出実装

完了。

- F-1: FIRE / OCEAN / STARFIELD 実装
- F-2: 5x8 ASCII テキスト描画
- F-3: クロスフェード
- F-4: LIVE / HEALING / ENTRY / OFF シーン
- F-5: 30 分安定性試験完走

### Phase G 以降

Phase G の基盤実装は進行中（約 70%）。

- TD UI 構築、HB 自動送信、再起動後の自動初期化まで実装済み
- 次は AudioLINKCORE 統合と外部入力による演出更新
- その後により大きい多ノード構成の足場作り
