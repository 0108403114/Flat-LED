### 2-2.5. 電源部の推奨選定メモ

| 部品 | 推奨候補 | 採用理由 | 補足 |
|---|---|---|---|
| PDシンクトリガ基板 | HUSB238 搭載 USB-C PD Sink Trigger Board | 12V 固定取得がしやすく、PD の上位電圧を安定して要求できる | CH224K 搭載基板でも代用可。ただし 12V 固定設定が明確なものを優先 |
| 降圧DC-DC | Pololu D24V50F5 | 12V → 5V で 3.5A 運用に余裕があり、固定 5V で扱いやすい | 安価な XL4015 系でも動作は可能だが、サイズと発熱で不利 |
| 代替降圧DC-DC | XL4015 5A Adjustable Buck Module | 入手性が高く安価 | 可変式なので 5.00V 調整必須。放熱対策を前提にする |

**採用方針**

- 第一候補は **HUSB238 搭載 PDシンクトリガ基板 + Pololu D24V50F5** の組み合わせ。
- 予算優先で試す場合のみ **HUSB238 or CH224K 基板 + XL4015 5A モジュール** を許容する。
- MVP での目標は 5V / 3.5A total なので、3A 級の降圧モジュール（LM2596、MP1584 系など）は採用しない。

### 2-2.6. 購入済み電源部の確認ポイント

- PD トリガ基板は、CIO バッテリー接続時に **本当に 12V が出るか** を最初にテスターで確認する。
- 降圧 DC-DC は、無負荷時に **5.0V 固定** であることを確認してから ESP32 をつなぐ。
- 降圧 DC-DC は 1A、2A、3A と段階的に負荷を上げ、発熱と電圧降下を確認する。
- 5V 母線の試験時は、先に抵抗負荷または USB 電流計を使い、いきなり LED マトリクスをつながない。

## 5. 最終電流予算（完成形 運用制限後）

| 消費元 | 電流（運用制限後） |
|---|---|
| メインノード全体 | 〜700mA |
| WS2812B 24LED イルミ @ 30%輝度 | 〜432mA |
| WS2812B 880LED マトリクス（FastLED 2A制限） | 2000mA |
| **合計** | **〜3.1A** |

35W バッテリーから 12V を取り出して 5V 化する構成であれば、この 3.1A 運用は成立範囲内。  
ただし LED は電流制限前提であり、フルホワイト常時点灯を前提にしないこと。  
5V/3.5A total を超える構成へ広げる場合は、5V 母線を 5A 以上へ拡張するか、LED枝を別電源化する。
